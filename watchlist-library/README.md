# GTF Watchlist Library

Host these files in the `gtf-autotrade-releases` repo and use the raw `index.json` URL inside the app's Watchlist Packs modal.

Default app URL:

```text
https://raw.githubusercontent.com/Ashbhati6/gtf-autotrade-releases/main/watchlist-library/index.json
```

Each pack is imported into a new folder. Existing folders are not merged or replaced.
